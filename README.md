## 어려웠던 점
1. decision - tree에 대한 코드 정보가 R에 비해 python이 적었다. 코딩 할때 구글링 해서 참고한 곳의 데이터와 나의 데이터가 달라서 데이터를 가공하는데 어려움이 있었다. 하지만, 구글링을 통해 내가 사용하려는 유병률 데이터와 최대한 비슷한 데이터를 쓴 코드를 참고하면서 수정하여 decision tree를 완성할 수 있었다.
